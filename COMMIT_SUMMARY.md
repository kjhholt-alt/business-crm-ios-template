# Initial Commit Summary

This repository contains the first iOS-first CRM template baseline.

Included:
- Expo Router app scaffold with 5 business tabs:
  - Dashboard
  - Accounts
  - Pipeline
  - Scanner
  - Connections
- Municipal CRM integrations:
  - Supabase REST access for customers/reminders
  - Railway scanner API stats/results
- Optional BarrelHouse integration:
  - Pipeline KPI endpoint hook
- Shared mobile theme and reusable UI primitives
- Phase roadmap in `IMPLEMENTATION_PLAN.md`
- Environment setup template in `.env.example`

Purpose:
- Unify municipal CRM functionality and BarrelHouse UX patterns into one reusable mobile business CRM product starter.
